Your data never leaves your browser

MarkOut processes everything locally using JavaScript. Your text is never sent to any server — not ours, not anyone else’s. The only external request is loading the Markdown parser from a CDN when the page first opens. After that, it works entirely offline. Disconnect from the internet and try it — it still works.

Frequently asked questions

What is MarkOut?

MarkOut is a browser-based tool that converts Markdown text to rich text and copies it to your clipboard in a format that Outlook (and most other email clients) renders correctly.

Paste Markdown in → MarkOut converts it → paste into Outlook with full formatting intact: bold, italics, bullet lists, numbered lists, headers, code blocks, tables.

Why would I need this?

AI assistants (ChatGPT, Claude, Copilot, Gemini) draft responses in Markdown. It looks great in the AI interface, but when you copy it and paste it directly into Outlook, the formatting disappears — you get raw syntax like **bold**, - bullet point, # Heading instead of the intended formatting.

MarkOut bridges that gap. It converts the Markdown to rich text so Outlook sees formatted content when you paste.

How do I use it?
  1. Optional: Check “Separate subject” if your content has a subject line you want to handle separately.
  2. Copy your Markdown text (from ChatGPT, Claude, wherever).
  3. Paste it into the left panel in MarkOut.
  4. The rich text is automatically copied to your clipboard.
  5. Switch to Outlook and paste. Done.

No button to click. Paste in, clipboard updates, paste out.

What is the “Separate subject” feature?

When checked, MarkOut looks for a subject line at the very top of your pasted content — either a Markdown heading (# My subject) or a plain-text email prefix (Subject: My subject).

If found, the subject appears in a pill in the header with its own Copy button, and it’s stripped from the rich text that goes to your clipboard. That way you can paste the body into Outlook first, then go back to MarkOut to copy just the subject line for the subject field.

Does it work with Gmail, Apple Mail, or other apps?

Yes. MarkOut writes rich text (HTML) to the clipboard, which any app that accepts formatted paste can read. It was designed with Outlook in mind but works with Gmail, Apple Mail, Word, Google Docs, Notion, and more.

Is it free?

Yes, completely free. No account, no subscription, no ads. MarkOut is an open-source tool hosted as a public service.

Can I self-host it?

Yes. MarkOut is a single HTML file with no build step and no backend. To self-host:

  1. Clone or download github.com/jonwadsworth/MarkOut
  2. Serve index.html from any static file host, or just open it locally in a browser

Note: the clipboard write API requires HTTPS or localhost, so file:// won’t work. Use a local server (python -m http.server) or deploy to any static host.
